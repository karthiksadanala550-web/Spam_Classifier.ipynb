# Spam Email Classifier

This project classifies messages as Spam or Not Spam using Machine Learning.

## Description

* Uses text data (SMS messages)
* Converts text into numbers
* Applies Naive Bayes algorithm

## Technologies

* Python
* Pandas
* Scikit-learn

## Output

* Predicts whether a message is spam or not
* Shows model accuracy
